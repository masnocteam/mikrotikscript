# mikrotikscript
Kliping kode skrip mikrotik untuk umum

## Try it

Just `download` and `import .rsc file` to your terminal router.

* **Example**
```
/tool fetch mode=http url="https://github.com/masnocteam/mikrotikscript/blob/main/mydynip.rsc"
```

```
/import mydynip.rsc
```
